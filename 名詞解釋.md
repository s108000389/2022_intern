## 品質管理概述
- 品質管理 : 監控製造過程中的變數(人/機/料/法)進而掌握品質變化狀況.
- 4M : 人/機/料/法
```
Man:作業者
Machine:機台
Material:材料
Method:作業方法
```
- 5M : 人/機/料/法/測量
```
Measurement:測量
```

## 雷射相關
### 機台組成
- EPC對邊機（Edge Position Control）是一種用於保持材料在生產過程中位置穩定的設備。它主要利用檢知電眼來偵測材料的邊緣位置，並通過控制推動器來調整材料的位置，確保材料在運作中始終保持在整齊正確的位置上
