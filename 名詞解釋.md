## 品質管理概述
- 品質管理 : 監控製造過程中的變數(人/機/料/法)進而掌握品質變化狀況.
- 4M : 人/機/料/法
```
Man:作業者
Machine:機台
Material:材料
Method:作業方法
```
- 5M : 人/機/料/法/測量
```
Measurement:測量
```

## 厚度單位
1 oz = 36 um  
1 mil = 25 um  
mm:um = 1:1000
