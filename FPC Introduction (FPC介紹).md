![image](https://user-images.githubusercontent.com/79491888/177457334-3272e299-0c8e-4340-b4d9-30b993657850.png)

### 接著劑配置  
製造條件:
```
氣壓,攪拌速度與時間,高度
```

品質確認:
```
1.黏度影響剝離強度:
  參照材料製造式樣書之黏度基準判定
  (依?cup?    穴徑之黏度計測定,
   黏度是否在規定秒數內)
2.接著劑配合後需靜置脫泡,接著劑中含氣泡
  會導致外觀不良
```

黏度計原理
```
特性檢查利用迴轉式黏度計測定黏度,此黏度計轉軸上方有一扭力計,依扭力讀值G換算成黏度;而現場配合室
黏度測定是利用流體流經一固定穴孔之CUP計算其時間(低黏度所需時間少,高黏度所需時間多,
時間規格由日本訂定),作為黏度依據
```

### 塗佈  
捲出
```
第一捲出:
基板膠片由此處捲出,確認材料編號、是否有外觀性不良、牽引方式

第二捲出:
貼合材放置處,銅箔或離形紙由此處捲出,需確認材料編號、牽引方式、張力
```

Coating
```
以供給機將接著劑由原液桶抽取至塗佈盤上，利用塗佈滾輪(AR)
及間隙滾輪(MR)調整接著劑厚度，使其達到所需的厚度標準
迴轉滾輪(BR)下壓將所需的接著劑附著於基板膠片(PI)上

注意事項:此工程最容易發生的不良為塗佈不均，而易造成塗佈不均
的原因主為供給機的供給量不足及塗佈盤太低

管理項目:AR、MR間隙與速度、BR間隙、BR寬幅、接著劑黏度(參照材料式樣書)

```

## 假本接著

### VGP
  VGP : Visual Guide pin Punch  
  利用CCD 鏡頭搜尋特定PATTERN作為定位點,利用模具作冲孔的動作,
  定義出設定孔,以利後續貼合及裁切定位(單軸VGP精度:±0.05mm;雙軸VGP精度:±0.02mm)  
  
  製造條件管理:
  ```
  捲出與捲取張力,程式條件
  ```
  
  品質管理項目:
  ```
  穴孔未打,擴孔,穴孔偏移,打折痕,銅表面傷痕,孔穴數量
  ```

### 假接著
  目的:將蝕刻或VGP打拔畢之製品貼上保護膠片或補膠膠片，藉由保護製品銅箔及強化製品物性，  
  避免銅箔直接與空氣接觸氧化或撓曲多次斷折，以達到製品絕緣及增加製品可撓性之目的。  
  廠內目前有手動與自動假接著兩種機型  
  ![image](https://user-images.githubusercontent.com/79491888/178403273-17075000-7b13-44ad-b8c5-fcc93850ef2a.png)  

  製造條件管理:
  ```
  壓合溫度,壓合秒數,壓合壓力,貼合手順
  ```
  
  品質管理項目:
  ```
  1.貼合偏移
  2.異物
  3.打折痕
  ```
  
### 本接著
  目的:製品經假接著時由治具設定使保膠初步黏附於製品上，但貼合效果仍未完全且有氣泡存在，  
  此時需以較高溫度、壓力使保膠之接著劑充分流動，且均勻分佈在銅箔表面上並使接著劑暫時硬化，  
  且藉由壓力使多餘氣體排出製品外。  
  
  ![image](https://user-images.githubusercontent.com/79491888/178403763-7cf07718-dae5-4ca6-a6f7-281f6e5f7da4.png)  
  
  製作條件管理:
  ```
  1.上下熱盤左中右溫度
  2.壓合壓力
  3.壓合秒數
  ```
  
  品質管理項目:
  ```
  氣泡,變色,接著劑溢出,附著物,打折痕
  ```

## 電鍍工程
  ![image](https://user-images.githubusercontent.com/79491888/178404224-a6e531ff-25a0-482d-8e55-f69bf5439b64.png)  
  
### 電鍍
  1.目的:在基材上鍍上金屬鍍層(deposit)，改變基材表面性質或尺寸。  
       例如賦予金屬光澤美觀、物品的防鏽、防止磨耗、提高導電度、潤滑性、  
       強度、耐熱性、耐候性、熱處理之防止滲碳、氮化 、尺寸錯誤或磨耗之零件修補。  
   
  2.FPC製程上將保膠未覆蓋所露出之銅利用電鍍法把金鍍於表層  
    台南廠目前電鍍方式分為兩種:  
    
    A.  
    自動電鍍,被鍍物置於掛架上,整支掛架放入鍍浴中進行電鍍  
      優點:鍍件直接面對陽極,因而鍍層均勻,且不因碰撞造成刮傷  
      缺點:費人工,且一個掛架只可放4SHEET,不可掛太多鍍件,否則會因電流差導致厚度差異  
      
    B.  
    連續電鍍,被鍍物成捲,經連續電鍍機台後,即電鍍完成

### 脫脂
  - 界面活性劑:溶液中加入界面活性劑，能使其表面張力立即減小,表面處理過程如 洗淨、脫脂、酸洗等界面活性劑被廣泛應用對表面處理之光澤化、平滑化，均一化都有相當幫助。
  - 廠內脫脂劑:橘酸(弱酸),160S(鹼),PAC-200(酸)  
  - 製造條件確認:脫脂槽溫度,比重(Be’),毛刷設定高度

### 微蝕刻
  - 目的: 去除銹垢或其他氧化物膜，要注意防止基材被腐蝕或產生氫脆。可加抑制劑以避免過度酸浸。酸浸完後要充份清洗。
  - 廠內使用:1.SPS(硫酸+G3+過硫酸鈉), 2.CPE-750(依蝕刻能力與產品狀況作區別,非依軟硬金區別,硫酸+雙氧水),3.Micro clean(D10前處理)
  - 製造條件確認:藥液濃度,溫度

### 活化
  - 目的:利用酸的腐蝕性,將預電鍍面腐蝕以增加表面粗糙度,促進鍍層附著性,可用各種酸溶液使銅表面活化,釋出銅離子
    在電鍍時能與電鍍金屬形成鍵結
  - 製造條件確認:酸液濃度,溫度

### 電鍍鎳
  1.經電刷使其預鍍面帶負電荷,在經過陽極板將金屬陽離子釋出,與預鍍面負離子結合,金屬離子即沉積於表面,完成電鍍  
  2.鎳電鍍之目的:鍍鎳是為了保護銅面，而鍍金是為了保護鎳層。因為金跟銅的密度差距相當度大，直接鍍金的話金跟銅會  
    產生“置換反應”金會跑進銅層中，而失去了表面處理的意義。所以說鎳層主要也是阻隔金跟銅，避免產生反應。  
  3.鍍金與鍍鎳使用之陽極板皆為白金鈦板屬不溶解性陽極,可控制金屬離子過度積集在鍍槽內  
  4.軟硬金所使用鎳塊成分不同  
  5.製造條件確認:藥液濃度,溫度,PH值  
  6.製造式樣確認:line 速度, 電流值,槽數  
  7.使用寬電刷(接觸面積大,效率高)  
  
  ![image](https://user-images.githubusercontent.com/79491888/178406928-02aed348-eac5-44d9-9873-bd90ef69191a.png)

  ``` 
  1.外觀:
       A.光澤度:胺基磺酸鎳光澤度比硫酸鎳高
       B.顆粒粗糙度:胺基磺酸鎳顆粒大小不平均
         硫酸鎳較平整
       C.直徑大小:胺基磺酸鎳中心圓直徑較小於硫酸鎳

  2.特性:
       A.胺基磺酸鎳,質軟(內應力低),電鍍時析出顆粒較大,
         電鍍後鍍層沉積鬆散,表面較粗糙
       B.硫酸鎳,質硬(內應力高),電鍍時析出顆粒較細,
         電鍍後沉積較密集,藥液中添加光澤劑,表面光澤度高
  ```
